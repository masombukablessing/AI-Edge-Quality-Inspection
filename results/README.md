Purpose: This folder contains the outputs of the AI-based Quality Inspection System. It stores the predictions made by the trained model for the input dataset. The results demonstrate how the system classifies products as good or defective.

1.Contents:
          predictions/ – Subfolder containing predicted images sorted by class:
          good/ → Images classified as good products.
          defective/ → Images classified as defective products.

2.CSV/Excel file (optional) – predictions.csv contains:
          filename → Name of the image
          predicted_class → Good or defective
          confidence_score → Model’s confidence in prediction.

3.Plots/Charts (optional) – Accuracy and loss graphs during model training.
4.Report (optional) – Summary of total good vs defective products and model performance.

Usage: Users can check this folder to see the results of the AI inspection system and analyze model predictions. It helps visualize the system’s performance and can be used for quality assurance reporting.
